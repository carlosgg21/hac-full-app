H.A.C. Renovation

Guía de Instalación Paso a Paso

Versión 1.0.0

¡Bienvenido a la Instalación!

Esta guía te ayudará a instalar el sistema H.A.C. Renovation en tu servidor local o de producción. Sigue los pasos en orden y no te tomará más de 15 minutos.

Tip: Si encuentras algún problema, revisa la sección de "Solución de Problemas" al final de esta página.

Requisitos del Sistema

Servidor Web

  • • Apache con mod_rewrite
  • • PHP 7.4 o superior
  • • Extensiones: PDO, PDO_MySQL, JSON

Base de Datos

  • • MySQL 5.7+ o MariaDB 10.2+
  • • phpMyAdmin (recomendado)

Recomendado: Laragon (Windows), XAMPP, WAMP o MAMP para desarrollo local.

1

Descargar o Clonar el Proyecto

Primero necesitas obtener los archivos del proyecto en tu computadora.

Opción A: Con Git (Recomendado)

git clone <url-del-repositorio> hac-renovation
cd hac-renovation

Opción B: Descargar ZIP

1. Descarga el archivo ZIP del repositorio
2. Extrae los archivos en una carpeta
3. Nombra la carpeta como "hac-renovation" (opcional)

2

Configurar el Servidor Web

Coloca el proyecto en el directorio correcto de tu servidor web.

Laragon (Windows - Recomendado)

  1. Abre Laragon
  2. Copia la carpeta del proyecto a: C:\laragon\www\
  3. Inicia Laragon (botón "Start All")
  4. Accede a: http://localhost/hac-renovation/

XAMPP / WAMP / MAMP

  1. Copia la carpeta del proyecto a htdocs (XAMPP) o www (WAMP/MAMP)
  2. Inicia Apache y MySQL desde el panel de control
  3. Accede a: http://localhost/hac-renovation/
3

Crear la Base de Datos

Necesitas crear la base de datos e importar la estructura.

Importante: Asegúrate de que MySQL esté corriendo antes de continuar.

3.1. Crear Base de Datos

Abre phpMyAdmin y crea una nueva base de datos:

Nombre: hac_renovation
Intercalación: utf8mb4_unicode_ci

3.2. Importar Schema

En phpMyAdmin:

  1. Selecciona la base de datos hac_renovation
  2. Ve a la pestaña "Importar"
  3. Haz clic en "Seleccionar archivo"
  4. Busca y selecciona: backend/database/schema.sql
  5. Haz clic en "Ejecutar"

O desde la línea de comandos: mysql -u root -p < backend/database/schema.sql

4

Configurar los Archivos

Edita los archivos de configuración con tus datos de base de datos.

¡Atención! Sin esta configuración, el sistema no podrá conectarse a la base de datos.

4.1. Editar database.php

Abre el archivo: backend/config/database.php

Cambia estos valores según tu configuración:

'host' => 'localhost',
'database' => 'hac_renovation',
'username' => 'root', // Cambia si es necesario
'password' => 'root', // Cambia si es necesario

4.2. Verificar config.php (Opcional)

El archivo backend/config/config.php generalmente no necesita cambios. Solo modifícalo si necesitas cambiar la zona horaria o el entorno (development/production).

5

Importar Datos de Prueba (Opcional)

Si quieres tener datos de ejemplo en el sistema para probarlo.

Este paso es opcional. Solo hazlo si quieres datos de ejemplo.

En phpMyAdmin:

  1. Selecciona la base de datos hac_renovation
  2. Ve a la pestaña "Importar"
  3. Selecciona: backend/database/fake_data.sql
  4. Haz clic en "Ejecutar"

O desde línea de comandos: mysql -u root -p hac_renovation < backend/database/fake_data.sql

6

Verificar la Instalación

¡Ya casi terminamos! Verifica que todo funcione correctamente.

Pasos de Verificación

  1. Abre tu navegador y ve a: http://localhost/hac-renovation/backend/
  2. Deberías ver la página de login o el dashboard
  3. Inicia sesión con las credenciales por defecto:
    • Usuario: admin
    • Contraseña: admin123

¡IMPORTANTE! Cambia la contraseña del administrador inmediatamente después de la primera sesión, especialmente si es un servidor de producción.

Solución de Problemas

Error: "No se puede conectar a la base de datos"

  • Verifica que MySQL esté corriendo
  • Revisa las credenciales en backend/config/database.php
  • Asegúrate de que la base de datos hac_renovation existe

Error: "404 Not Found" o las rutas no funcionan

  • Verifica que mod_rewrite esté habilitado en Apache
  • Revisa que el archivo backend/.htaccess exista
  • En Laragon, reinicia Apache

La página muestra código PHP en lugar de ejecutarlo

  • Verifica que PHP esté instalado y corriendo
  • Revisa la configuración de Apache para archivos PHP
  • En Laragon, reinicia Apache

Problemas con caracteres especiales (acentos, ñ)

  • Asegúrate de que la base de datos use utf8mb4
  • Verifica que los archivos PHP estén guardados en UTF-8
  • Revisa la configuración de charset en database.php

¡Instalación Completada!

El sistema está listo para usar. ¡Disfruta de H.A.C. Renovation!
